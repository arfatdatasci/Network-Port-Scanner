# Network Port Scanner

A simple Network Port Scanner built using Python that scans a target system and detects open TCP ports.

## Features
- Scan ports from 1 to 1024
- Detect open ports on a target host
- Uses multithreading for faster scanning
- Simple command-line interface

## Technologies Used
- Python
- Socket Programming
- TCP/IP Networking
- Multithreading

## Installation

Clone the repository:

git clone https://github.com/yourusername/network-port-scanner.git

Move into project directory:

cd network-port-scanner

Run the program:

python3 port_scanner.py

## Example

Enter target IP or domain:

scanme.nmap.org

Output:

Port 22 : Open  
Port 80 : Open
